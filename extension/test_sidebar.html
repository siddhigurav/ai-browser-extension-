Testing Instructions

This page helps you test the Sider AI-style sidebar functionality.

Step 1: Load Extension

Make sure the extension is loaded in Chrome as an unpacked extension.

Step 2: Open Sidebar

Try one of these methods to open the sidebar:

  • Press Alt+S on your keyboard
  • Right-click anywhere on this page and select "Toggle AI Assistant Sidebar"
  • Click the extension icon in the toolbar (then use Alt+S)

Step 3: Test Features

Once the sidebar opens, test its features:

  • Try the chat functionality
  • Test the summarization features
  • Try explaining code or text
  • Process PDFs and images with OCR

Important Notes:

  • The sidebar should appear on the right side of your browser window
  • It's designed to be non-intrusive and shouldn't affect the layout of the webpage
  • All processing happens securely with your API keys
  • Press Alt+S again to toggle the sidebar visibility
