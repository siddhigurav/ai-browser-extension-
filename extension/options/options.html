🔑 API Key Setup:
This extension uses OpenRouter for free AI models. You need an API key to use the extension.
  1. Visit https://openrouter.ai/keys
  2. Sign up (free) and create an API key
  3. Paste your key below and click "Save key"
  4. Start using free models like Llama 3, Mistral, and Gemma!
💡 Free tier includes generous credits. No credit card required for basic usage.

API Key Configuration

Your API key is stored securely in your browser using Chrome's secure storage. It is never shared with anyone except the AI provider you selected.
